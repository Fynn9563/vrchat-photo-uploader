ℹ️ About VRChat Photo Uploader

VRChat Photo Uploader

Version Loading...

A powerful tool for uploading VRChat photos to Discord with intelligent grouping, metadata extraction, and automatic compression when needed.

Features:

  • 🏷️ Automatic metadata extraction from VRChat photos
  • 📋 Smart grouping by world and players
  • 🗜️ Automatic compression only when Discord requires it
  • 🔄 Retry failed uploads with exponential backoff
  • 📊 Upload statistics and progress tracking
  • 🎨 Dark/Light theme support
  • ⚡ Global keyboard shortcuts
  • 💬 Discord Forum channel support
  • 👁️ Ctrl+Hover image previews in upload queue

Keyboard Shortcuts:

Ctrl + Shift + U Upload files
Ctrl + Hover Preview image thumbnails
