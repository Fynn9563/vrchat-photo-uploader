ℹ️ About VRChat Photo Uploader

📸

VRChat Photo Uploader

Version Loading...

A powerful tool for uploading VRChat photos to Discord with intelligent grouping, metadata extraction, and automatic compression when needed.

Features:

  • 🏷️ Automatic metadata extraction (VRCX & VRChat native)
  • 📋 Smart grouping by world, time, and players
  • 🗜️ Tiered compression fallback system
  • 🔄 Background auto-upload with folder watching
  • 📊 Real-time upload progress with ETA
  • 🎨 Dark/Light theme support
  • ⚡ Global keyboard shortcuts
  • 💬 Discord Forum channel & single thread mode
  • 👁️ Ctrl+Hover image previews
  • 🔀 User webhook overrides per player

Keyboard Shortcuts:

Ctrl + Shift + U Upload files
Ctrl + Hover Preview image thumbnails
